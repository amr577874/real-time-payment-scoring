Real-Time Fraud Detection with Kafka & Python

This project demonstrates a complete real-time fraud detection pipeline using:
Apache Kafka for streaming 
Python for data generation & model inference
Scikit-learn for training a Random Forest fraud detection model
KafkaAdminClient for automatic topic management
The pipeline generates synthetic financial transactions, streams them to Kafka, consumes them, and runs fraud predictions in real time.

the other way there are 4 .py files to run with python directly

├── model.py                 # Trains ML model on synthetic data
├── Cosumer.py              # consume from kafka topic - runs real-time predictions - store the data in mysql db

# Requirements
installed kafka v 2.13-3.7.0.
python 3.11.9
mysql 9.5.0 

How to run the app
1 - start  Zookeeper & Kafka 
  bin\windows\zookeeper-server-start.bat config\zookeeper.properties
  bin\windows\kafka-server-start.bat config\server.properties
  
2 - run .py files in order as below in bash
  python createkafkatopic.py
  python KAFKAProd.py
  python model.py
  python Cosumer.py
